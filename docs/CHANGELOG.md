# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-16

### Added
- Initial release of OMEGA Warehouse Planner
- 2D top-down warehouse layout view with pan and zoom
- 3D isometric visualization with bundle rendering
- 5 layout strategies (Type Split, Compact, FIFO, Quick Access, Project Cluster)
- Space utilization heatmap and analytics
- Truck loading planner with trip optimization
- PDF report export (A3 landscape)
- CI/CD pipeline with GitHub Actions
- Automatic deployment to GitHub Pages
